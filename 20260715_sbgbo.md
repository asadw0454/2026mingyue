<h2>正文</h2>
<p>微软（MSFT）周二宣布，将下调两款 Xbox Game Pass 订阅套餐的价格，这一举措与整个行业的大趋势背道而驰。</p>
<p>微软表示，其 Game Pass Ultimate（主机 + PC 全能版）订阅费将从每月 29.99 美元降至 22.99 美元，PC 版 Game Pass 则从 16.49 美元降至 13.99 美元。</p>
<p>Game Pass 基础版与高级版价格保持不变。</p>
<p>不过该公告附带一项附加条款：微软旗下热门的《使命召唤》系列游戏，今后将不再在发售首日登陆订阅服务，而是在首发约一年后才加入 Game Pass。</p>
<p>已发售的《使命召唤》旧作不受此次调整影响。</p>
<p>此次下调 Game Pass Ultimate 与 PC 版 Game Pass 价格，是对去年 10 月涨价政策的逆转。</p>
<p>当时微软上调了这两项服务价格，遭到玩家批评。</p>
<p>彼时，微软将 Ultimate 订阅费从每月 19.99 美元大幅上调 50% 至 29.99 美元，PC 版 Game Pass 则从 11.99 美元涨至 16.49 美元。</p>
<p>此次调价是微软游戏业务新任 CEO 阿莎・夏尔马上任后的首批重大决策之一。</p>
<p>深受游戏社区喜爱的前 CEO 菲尔・斯宾塞于今年 2 月卸任，夏尔马接任该职位，此举大概率将为她赢得玩家好感。</p>
